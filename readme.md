## Project Structure

The project is organized by day, with each day having its own module. Each module contains the solution for both parts of the puzzle, as well as the input file.

```
src
├── day1
│   ├── input.txt
│   ├── mod.rs
│   ├── part1.rs
│   └── part2.rs
├── day2
...
```
